# drf-tinyurl
Basic DRF API for URL Shortener.

## How to run
To run app, download it and run `docker compose up -d` command.
It runs application on your localhost, you can saw it on http://localhost/ address.

## Documentation
If your app is on, you can open `http://localhost/docs/` address where 
you have API documentation where all available endpoints are listed.

## Tests
If your app is on, you can run all tests on separated container using `docker compose run -rm test` command.
