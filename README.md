# drf-tinyurl
A basic DRF API for a URL shortener.

## How to run
Clone or download the repository, then start the application by running:

`docker compose up -d`

The application will be available at:

http://localhost/

## Documentation
Once the application is running, you can access the API documentation at:

http://localhost/docs/

The documentation lists all available API endpoints and allows you to interact with them.

## Tests
To run the test suite, execute:

`docker compose run --rm test`

This command runs all tests in a separate container.
